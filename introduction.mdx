## About ZOODOM

ZOODOM is a production-ready website designed to improve usability, accessibility, and organic visibility for the National Zoological Park. The site integrates key features including species profiles, event calendars, visit planning tools, educational blog content, and multimedia resources.

## Technology Stack

ZOODOM is built with modern web technologies:

- **Astro** - Fast, content-focused static site framework
- **Bootstrap 5.3.8** - Responsive CSS framework for layout and components
- **HTML/CSS/JavaScript** - Core web technologies for structure and interactivity
- **SEO Optimized** - Meta tags, Open Graph, and semantic HTML for search visibility
- **Accessibility** - WCAG compliant markup and ARIA attributes

## Getting Started

<Steps>
  <Step title="Clone the repository">
    Clone the ZOODOM repository to your local machine:
    ```bash
    git clone https://github.com/Iceheop/zoodom.git
    cd zoodom
    ```
  </Step>
  
  <Step title="Install dependencies">
    Install the required npm packages:
    ```bash
    npm install
    ```
  </Step>
  
  <Step title="Start the development server">
    Launch the local development server:
    ```bash
    npm run dev
    ```
    Your site will be available at `http://localhost:4321`
  </Step>
</Steps>

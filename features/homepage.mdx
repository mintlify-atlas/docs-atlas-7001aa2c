The ZOODOM homepage serves as the main entry point for visitors, featuring a welcoming hero section with a call-to-action for reservations.

## Overview

The homepage is built with Astro and Bootstrap 5.3.8, providing a responsive and modern user experience. It introduces visitors to the national zoo park and encourages them to plan a family visit.

## Current Implementation

The homepage is located at `src/pages/index.astro` and includes:

## Responsive Design

Bootstrap's responsive grid system ensures the homepage works across all devices:

- **Desktop (lg):** Two-column layout with text on left, image on right
- **Tablet (md):** Maintains two-column layout with adjusted spacing
- **Mobile:** Single column stack with full-width content

<Note>
  The navbar uses Bootstrap's `.navbar-expand-lg` class to collapse navigation into a hamburger menu on smaller screens (though the toggle button isn't currently implemented in the code).
</Note>

## Development

To run the homepage locally:

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

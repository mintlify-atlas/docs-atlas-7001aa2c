The ZOODOM color system is built using CSS custom properties defined in `src/assets/global.css`. All colors are accessible via CSS variables throughout the application.

| `--color-brand-green-800` | `#246C32` | Deep green accent |
| `--color-brand-green-700` | `#388E49` | Primary buttons, links |
| `--color-brand-green-600` | `#51B063` | Hover states |
| `--color-brand-green-500` | `#6DD281` | Primary brand color |
| `--color-brand-green-400` | `#8EF4A2` | Highlights |
| `--color-brand-green-300` | `#ACFFBC` | Light accents |
| `--color-brand-green-200` | `#C4FFD0` | Subtle backgrounds |
| `--color-brand-green-100` | `#DCFFE3` | Very light backgrounds |
| `--color-brand-green-50` | `#F4FFF6` | Lightest tint |

```css

  background-color: var(--color-brand-green-50);
```

## Color Accessibility

When using colors, ensure sufficient contrast ratios:

- **Large text (18px+)**: Minimum 3:1 contrast ratio
- **Normal text**: Minimum 4.5:1 contrast ratio
- **UI elements**: Minimum 3:1 contrast ratio

### Recommended Combinations

```css
/* High contrast - excellent readability */
.high-contrast {
  color: var(--color-neutral-black-900);
  background-color: var(--color-neutral-white-900);
}

/* Brand colored text on light background */
.brand-text {
  color: var(--color-brand-green-700);
  background-color: var(--color-brand-green-50);
}

/* White text on dark background */
.dark-section {
  color: var(--color-neutral-white-900);
  background-color: var(--color-brand-green-900);
}
```

## Implementation

All color variables are defined in `/src/assets/global.css:1-58`:

```css
:root {
  /* Primitives */
  
  /* Green */
  --color-brand-green-900: #144A1F;
  --color-brand-green-800: #246C32;
  /* ... etc */
}
```

These variables are globally available after importing the CSS file in your layout or component.
